# remindian

Automated reminder manager sourced from Obsidian check lists (initial scaffold).

## Development

Build the project:
```
make build
```

Run the CLI (optionally pass a name):
```
make run ARGS="Alice"
```

Run tests:
```
make test
```

Clean build artifacts:
```
make clean
```
